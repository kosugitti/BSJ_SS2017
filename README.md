## このセミナーの目標

### 考え方の目標（伝えたいこと）

1. 「与えられたデータの特徴を探る」研究から「データを作り出すメカニズムを考える」研究へ変わること。
2. 求めるものが確率変数になること。
3. サンプルから作られた事後分布の近似データを解釈することにより間違いが生じにくくなること。

### 技術的な目標（学んで欲しいこと）

1. MCMCサンプルの読み方
2. 仮装データの組み立て方
3. 添字の使い方（階層性への自然な拡張）
